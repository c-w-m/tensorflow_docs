#### Usage:



```
>>> model = tf.keras.Sequential([
...     tf.keras.layers.Dense(5, input_shape=(3,)),
...     tf.keras.layers.Softmax()])
>>> try:
...   import yaml
...   config = model.to_yaml()
...   loaded_model = tf.keras.models.model_from_yaml(config)
... except ImportError:
...   pass
```
