<!-- Placeholder for "Used in" -->

The returned tensor uses the memory shared by dlpack capsules from other
framework.

  ```python
  a = tf.experimental.dlpack.from_dlpack(dlcapsule)
  # `a` uses the memory shared by dlpack
  ```
